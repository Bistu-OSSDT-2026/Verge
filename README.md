## 🎮 游戏简介

**Verge** 是一款融合**时间循环机制**与**角色部署策略**的 2D 塔防游戏。玩家守护神秘的「钟摆核心」，在白天部署角色发展经济，黄昏备战，夜晚抵御怪潮，最终在黎明时刻见证全屏清敌的极致爽感——然后一切重新开始，迎接更艰难的明天。

核心玩法参考了《植物大战僵尸》的路径塔防与《明日方舟》的角色部署与阻挡策略，创新点在于**昼夜循环**带来的节奏切换：白天经营、夜晚死守、黎明清屏。

---

## ✨ 核心特色

- **时间循环**：白天(90s) → 黄昏(30s) → 夜晚(90s) → 黎明(清屏) → 下一天，3天为一局。
- **角色部署**：先锋、重装、狙击三类单位分工明确，兼顾前线挡敌、远程输出与经济铺设。
- **阻挡机制**：近战角色可阻挡敌人并形成排队显示，死亡、撤回、黎明清屏时都会正确释放。
- **撤回系统**：已部署角色可通过左键点格子弹出确认面板，返还一半费用后撤离。
- **黎明清屏**：全屏白闪、敌人消散、角色回血，让整局节奏形成强烈的阶段切换。
- **经济策略**：金矿白天与黄昏产出、夜晚停产，结合击杀赏金形成资源节奏。
- **像素动画**：AnimatedSprite2D 角色/敌人动画，配合暗色 UI 与诗意结算文案。
- **音效系统**：BGM + SFX 已接入菜单、部署、攻击与按钮交互。

## 🕹️ 操作指南

### 游戏流程
1. **白天（90秒）**：点击底部角色按钮 → 选中角色 → 点击地图部署位 → 放置金矿积累金币
2. **黄昏（30秒）**：少量敌人出现，检查防线，调整阵型
3. **夜晚（90秒）**：大量敌人进攻，角色自动攻击并**阻挡**敌人，金矿停产
4. **黎明（3秒）**：全屏白闪，敌人清屏，角色回血，进入下一天
5. **Day 3 夜晚结束**：核心未被摧毁 → 胜利！

### 操作按键

| 按键 | 功能 |
|------|------|
| 鼠标左键 | 选择角色 / 部署角色 / 点击已部署角色弹出撤回面板 / 点击按钮 |
| `Esc` | 打开/关闭暂停菜单 |
| 倍速按钮 | 1x ↔ 2x 游戏速度切换 |
| `F3` | 测试：直接触发胜利结算 |
| `F4` | 测试：直接触发失败结算 |
| `R` | 失败结算面板：重新开始 |

### 阻挡机制

角色部署后会自动阻挡进入其格子的敌人：
- **先锋**：可阻挡 2 个敌人，中规中矩的前线单位
- **重装**：可阻挡 4 个敌人，堵路核心
- **狙击**：高台远程，不参与阻挡

被阻挡的敌人会沿路径方向排队显示，前排死亡后后排自动补位。敌人被阻挡时会持续攻击阻挡者。

### 撤回机制

左键点击已部署角色所在的格子 → 弹出撤回确认面板 → 点击「撤回」返还一半部署费用，释放被阻挡的敌人。

### 暂停菜单
- **继续游戏** — 返回游戏
- **设置** — 音效/音乐设置（开发中）
- **返回章节选择** — 二次确认后跳转
- **返回主菜单** — 二次确认后跳转

## 🔧 开发说明

### 代码规范
- 所有脚本使用 **GDScript**，命名风格 `snake_case`
- 节点查找优先使用 `find_child()`，避免 `$` 路径硬编码
- 信号通过 `SignalBus` 单例传递，模块间不直接依赖

### 关键技术注意
- **暂停机制**：暂停菜单用 `get_tree().paused = true`，胜利/失败面板用 `Engine.time_scale = 0`
- **Tween 陷阱**：`Engine.time_scale = 0` 时 Tween 完全不推进，UI 状态须立即设置最终值
- **动画信号**：`AnimatedSprite2D.animation_finished` 不能稳定承担关键逻辑，角色/敌人攻击与死亡逻辑已改为解耦或定时兜底
- **鼠标拦截**：`visible = false` 的 Control 仍可能拦截鼠标，需同步设 `mouse_filter = IGNORE`
- **阻挡与释放**：敌人被阻挡后会排队显示，角色死亡/撤回/黎明清屏时必须正确释放阻挡队列，避免引用残留
- **撤回面板**：左键点击已部署格子触发，格子在部署模式下会跳过撤回检测，避免误触

### 测试快捷键（发布前删除）
- `F3` — 直接触发胜利结算面板
- `F4` — 直接触发失败结算面板
